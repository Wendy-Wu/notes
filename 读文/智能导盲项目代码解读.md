    - oakd_ppes.py

      - 别人的工作，语音识别。

        ```python
        speech = SpeechRecognizer()
        speech.start()
        ```

      - 跑的是如下pipeline：

      - ```python
        ppesPipeline.run(speech)
        speech.listen = False
        speech.end = True     
        speech.join()  
        ```
